# geol3600_eqcat
Earthquake Catalog for GEOL3600


Possible Tasks: 
1. Create data model, defensive programming for earthquake objects
2. Key functions: filter by bounding box, filter by time, or add up moments
3. Read USGS data file: https://earthquake.usgs.gov/earthquakes/map/
4. Read SCEDC data file: https://scedc.caltech.edu/data/eq-catalogs.html 
5. Write data file in some preferred format
6. Plot map
7. Plot Lollipop time series plots
8. Write tests
